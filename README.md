# Entrega Chapp
En esta entrega figuran los siguientes archivos:
- Chapp.zip: toda la carpeta del proyecto de Android Studio (build clean con código comentado).
- Demos: 2 vídeos del primer funcionamiento de la aplicación y el resultado final.
- README: este mismo documento explicando la entrega.
- APK: aplicación compilada lista para instalar.
- Presentación: está a medias por que no me ha dado tiempo a acabarla, pero en GitHub se actualizará.

## Especificaciones
Este proyecto de ha probado con dos móviles Android:
- Readmi 13 pro+ 5G con Android 15 (API 35)
- Mi A1 con Android 9 (API 28)

El SKD mínimo es para el API 28. La aplicación se ve bien en los dos móviles, sin ningún problema de visualización por las diferentes dimensiones de la pantalla. La IP que se ha probado era 192.168.1.37 correspondiente al teléfono más nuevo (el Readmi).

Ante cualquier problema se puede consultar el repositorio del proyecto en este enlace de [GitHub](https://github.com/solsolet/chApp.git).
